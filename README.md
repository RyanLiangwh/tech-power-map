# Tech Power Map · 全球科技权力图谱

一张可交互的科技商业人物关系网络图:**187 位人物 · 455 条关系**,横跨工业时代到 AI 时代(洛克菲勒 → 八叛徒 → 乔布斯 → 奥特曼)。

**在线访问**:启用 GitHub Pages 后访问 `https://<你的用户名>.github.io/tech-power-map/`

## 功能

- 密集网状力导向图,缩放/拖拽/搜索(中英文)
- 点击人物:查看其全部关系(家人/创业搭档/师徒/投资/竞争/恩怨…)
- 点击连线:查看两人关系详情
- 双人关系查询:任选两人,自动找出最短关系链
- ✦ 隐藏关系:21 条精选反直觉洞察 + 随机挖掘跨阵营隐藏连接
- 时代筛选:工业与电气 / 晶体管 / PC / 互联网 / 移动云 / AI
- 人物头像:自动从 Wikipedia 加载(失败回退为彩色节点)

## 使用

直接用浏览器打开 `index.html` 即可,无需构建。依赖 D3.js(CDN)。

## 数据说明

人物与关系基于公开报道整理,信息时点约为 2026 年中;关系描述含主观概括,仅供参考与娱乐,欢迎提 PR 修正。
